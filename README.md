# 美容院管理系统 python423

## 项目概述

美容院管理系统是基于Python和Vue开发的一个前后端分离的项目，主要服务于大学生课程设计作业，旨在帮助学生们学习和掌握Python编程技能，同时提高他们的项目设计与开发能力。

## 技术栈

- 后端：Python 3.8 + Django 3.2
- 前端：Javascript + Vue
- 数据库：MySQL 5.7
- 开发平台：Pycharm + vscode
- 运行环境：Windows 10/11

## 功能模块

### 系统管理

- 服务管理：录入、修改、查询服务基本信息（名称、简介等）
- 类型管理：管理服务类型信息（名称等）
- 美容师管理：管理美容师信息
- 用户管理：管理用户信息，包括新增、编辑和删除
- 轮播图管理：发布和管理轮播图
- 评论管理：用户发布评论，管理员维护评论

### 用户操作

- 注册登录：用户注册和登录
- 门户浏览：用户可浏览服务列表（最新、最热）
- 热门推荐：基于点击率排序算法的服务推荐
- 用户中心：修改用户资料、管理密码、收藏点赞等
- 我的预约：查看和管理预约信息
- 意见反馈：用户提交反馈
- 模糊搜索：支持服务信息的模糊搜索
- 服务评论：用户可在服务详情页发表评论

## 系统角色

- 管理员：管理系统信息，维护用户评论
- 用户：注册、登录、预约服务、发表评论等

## 运行环境

- 操作系统：Windows 10/11
- 后端环境：Python 3.8，Django 3.2，MySQL 5.7
- 前端环境：现代浏览器支持

## 目录结构

```
项目根目录/
├── 后端代码（Django项目）
├── 前端代码（Vue项目）
└── 数据库配置及脚本
```

## 核心亮点

- 前后端分离的技术架构，便于分工开发和维护
- 采用MySQL数据库，保障数据持久性和安全性
- 支持模糊搜索，方便用户快速定位服务信息
- 热门推荐算法，提高用户服务选择满意度
- 细致的权限管理，确保系统运行安全稳定

## 部署步骤

1. 准备Python 3.8环境，安装Django 3.2框架
2. 安装并配置MySQL 5.7数据库
3. 部署后端代码至服务器，配置环境
4. 部署前端代码至Web服务器，配置访问路径
5. 运行数据库脚本，初始化数据
6. 启动前后端服务，进行测试
